The basis of the software is to help college students, or those who are in need of food resources
to find resources that fit their needs. The software will allow for the user to create an account,
with that step they are able to unlock all of the features provided. Some features will include:
being able to search for specific foods, recipes, ingredients, ect. Another feature of the software
is that users will be able to follow other users, befriend them, and even be able to chat with 
one another. Users will also be able to share photos and videos of recipes. Additional features 
will also be that users will be able to search for economical places to shop for ingredients, 
and/or even look for events/drives where food can be found free of charge.
